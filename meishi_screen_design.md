# 名刺管理アプリ 画面設計書

作成日：2026年3月28日

---

## 設計方針

- **スマホファースト**: メイン用途がスマホ撮影のため、375px幅を基準に設計
- **レスポンシブ**: PC版はBootstrap5のブレークポイントで拡張（768px以上で2カラム化）
- **PWA対応**: manifest.json + サービスワーカーでホーム画面追加可能に

## 各画面の詳細設計

### 1. ログイン画面 (/login)

**レイアウト**: センタリングされたカード型フォーム

| 要素 | 仕様 |
|---|---|
| メールアドレス | text input, required |
| パスワード | password input, required |
| ログインボタン | submit |
| エラー表示 | フォーム上部にアラート |

※ユーザー登録画面は管理者がアクセスする /admin/users から

---

### 2. 名刺一覧 (/）

**スマホ版**:
- 上部: 検索バー（会社名・氏名・電話番号でインクリメンタルサーチ）
- フィルター: タブ型（全て / 社内共有 / 自分のみ）
- リスト: アイウエオ順セクションヘッダー付き
  - 各行: アバター（イニシャル）+ 氏名 + 会社名 + 公開バッジ
- FAB: 右下の＋ボタンで名刺登録へ

**PC版（768px以上）**:
- 左サイドバー: ナビゲーション（一覧/会社/バッチ/CSV/ユーザー管理/設定）
- メインエリア: 検索＋フィルター＋カードグリッド or テーブル表示切替
- 会社グルーピング表示: 会社名でアコーディオン展開

---

### 3. 名刺登録 (/cards/new)

**スマホ版**:
- 表面画像: カメラ起動ボタン + ギャラリー選択ボタン
  - `<input type="file" accept="image/*" capture="environment">` でカメラ直接起動
- 裏面画像: オプション。「＋裏面を追加」ボタンで表示
- 公開設定: セレクトボックス（社内共有 / 自分のみ）
- タグ: テキスト入力（カンマ区切り or タグ候補サジェスト）
- OCR開始ボタン: 押下でアップロード → R2保存 → Vision API → Claude API

**PC版**:
- ドラッグ＆ドロップエリア + ファイル選択ボタン
- プレビュー表示

---

### 4. OCR確認 (/cards/confirm)

**共通**:
- 上部: 名刺画像プレビュー（タップで拡大）
- OCRステータス: 処理中 → 完了 の表示
- フォーム: 自動抽出された各項目を編集可能に表示
  - 会社名（日本語/英語）
  - 部署名
  - 役職
  - 氏名（漢字/フリガナ/ローマ字）
  - 電話番号 × N（種別セレクト付き、「＋追加」ボタン）
  - メール × N（種別セレクト付き、「＋追加」ボタン）
  - 資格・肩書き × N（「＋追加」ボタン）
  - 郵便番号 + 住所 + 建物名
  - Webサイト
  - SNS情報
  - 裏面情報（事業内容メモ / 拠点情報）→ テキストエリア
  - メモ
- ボタン: 保存 / キャンセル

**会社名の自動マッチング**:
- 会社名入力時にDB既存のcompaniesをサジェスト
- 一致する場合は自動でcompany_idをリンク
- 新規の場合はcompanies新規作成

---

### 5. 名刺詳細 (/cards/<id>)

**レイアウト**:
- 画像エリア: 表・裏をサムネイル表示（タップで拡大モーダル）
- 基本情報: アバター + 氏名 + 役職
- 会社情報: 会社名 + 部署（会社名タップで同一会社の名刺一覧へ）
- 連絡先: 電話番号（タップで発信）、メール（タップで送信）
- 住所: Google Mapsリンク付き
- 資格・肩書き: バッジ表示
- タグ: バッジ表示
- メモ: テキスト表示
- 裏面情報: アコーディオン展開
- フッター: 登録日 + 登録者 + 公開設定バッジ

**アクション**:
- 編集ボタン → /cards/<id>/edit
- 削除ボタン（確認ダイアログ付き）

---

### 6. 名刺編集 (/cards/<id>/edit)

- OCR確認画面と同じフォーム構成
- 既存データをプリフィル
- 画像の差し替え可能

---

### 7. 会社一覧 (/companies)

- 会社名一覧（名刺件数付き）
- 会社名タップ → その会社の名刺一覧を展開
- 統合機能: チェックボックスで2社選択 → 「統合」ボタン
- 統合解除: 統合済み会社に「解除」ボタン表示

---

### 8. バッチアップロード (/batch)

**PC専用画面**:
- ドラッグ＆ドロップで複数ファイルアップロード
- ファイル一覧: ファイル名 + ステータス（待機中/処理中/完了/エラー）
- 一括OCR開始ボタン
- 進捗バー: processed_count / total_count
- 完了した名刺はOCR確認画面へのリンク

---

### 9. CSV管理 (/csv)

**エクスポート**:
- フィルター: 全件 / 自分のみ / 社内共有
- ダウンロードボタン（UTF-8 BOM付きCSV）

**インポート**:
- ファイル選択
- プレビュー表示（先頭5行）
- カラムマッピング確認
- インポート実行

---

### 10. ユーザー管理 (/admin/users) ※管理者のみ

- ユーザー一覧: 名前 + メール + 権限 + 登録日
- 新規ユーザー登録フォーム
- パスワードリセット機能

---

### 11. 設定 (/settings)

- パスワード変更
- 表示名変更

---

## PC版レイアウト方針

```
┌─────────────────────────────────────────┐
│  ヘッダー（ロゴ + ユーザー名 + ログアウト）    │
├────────┬────────────────────────────────┤
│        │                                │
│  サイド  │     メインコンテンツエリア        │
│  バー   │                                │
│  200px  │     名刺一覧: カードグリッド      │
│        │     詳細: 左に画像、右に情報      │
│  ・一覧  │     編集: 2カラムフォーム         │
│  ・会社  │                                │
│  ・バッチ │                                │
│  ・CSV  │                                │
│  ・ユーザー│                               │
│  ・設定  │                                │
│        │                                │
└────────┴────────────────────────────────┘
```

- 768px以上: サイドバー表示、メイン領域は2カラム活用
- 768px未満: サイドバー非表示、ハンバーガーメニュー、1カラム
